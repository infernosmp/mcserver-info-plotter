Minecraft Server Info Plotter

Visualising Minecraft server data from CSV files

Project Overview

This project reads Minecraft server data from a CSV file and plots the top 10 servers by player count, grouped by country, using Python and Matplotlib.

  • Input: export.csv with server data
  • Output: bar chart of top 10 countries by player count
  • Tech stack: Python, Matplotlib, CSV

How It Works

  1. Loads export.csv from the project directory.
  2. Extracts country (column 11) and player count (column 5).
  3. Sorts servers by player count in descending order.
  4. Displays a bar chart of the top 10 entries.

Getting Started

Basic steps to run the script:

  1. Install Python 3 and Matplotlib.
  2. Place export.csv next to mcserverinfo.py.
  3. Run the script from PowerShell:
py -m pip install matplotlib
py mcserverinfo.py
